# Developer Portfolio

This is a modern developer portfolio web application built with React, Vite, TailwindCSS, and TypeScript.

## Getting Started

### Prerequisites

Make sure you have Node.js installed.

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the application locally in development mode:
   ```bash
   npm run dev
   ```

3. Build the application for production:
   ```bash
   npm run build
   ```
